1. 네비게이션 바 기본 설정 변경

현재 문제





플레이어가 기본으로 표시되어 다른 메뉴로 이동 불가

수정 요청





미니 플레이어: 기본값(default)을 숨김(hide) 상태로 설정



하단 메뉴바: 기본값을 **항상 표시(노출)**로 설정



곡 재생 시에만 미니 플레이어가 나타나도록 변경

2. 로고 크기 확대

현재 문제





로고가 화면에서 거의 보이지 않을 정도로 작음

수정 요청





화면 비율에 맞춰 로고 크기를 더 확대해주세요



모바일, 태블릿, 데스크톱 등 다양한 화면 크기에서 적절히 보이도록 반응형* 설정

3. 비로그인 서비스 이용 허용

현재 문제





로그인해야만 서비스 사용 가능

수정 요청

비로그인 상태에서 사용 가능한 기능:





✅ 음악 검색



✅ 음악 재생



✅ 재생 목록 관리 (세션* 내에서만)

로그인이 필요한 기능:





플레이리스트 저장 (영구 보관)



저장된 플레이리스트 불러오기

구현 방식:





앱 실행 시 로그인 화면을 건너뛸 수 있도록 "로그인 없이 계속하기" 옵션 추가



비로그인 사용자가 플레이리스트 저장 시도 시 로그인 유도 메시지 표시

4. Up Next 자동 추천 기능 구현

수정 요청

참고 앱: YouTube Music

Up Next 목록 구성:





현재 재생 중인 곡과 유사한 음악/영상 자동 검색



검색 결과 중 약 20개를 Up Next 목록에 자동 추가



추천 기준: 같은 아티스트, 유사 장르*, 관련 곡 등

동작 방식:





곡 재생 시 자동으로 Up Next 목록 생성



사용자가 수동으로 곡을 추가할 수도 있음

5. 검색 기능 메뉴별 분리

현재 문제





프로필 메뉴에 불필요한 검색 기능이 있음



각 메뉴의 검색 목적이 다름

수정 요청

메뉴별 검색 기능 정의:
